Ambiente de pruebas para mi aplicación Streamlit con IA
# Chat IA para Atención Estudiantil

Proyecto de investigación desarrollado como parte de la Maestría en Inteligencia Artificial.

## Objetivo

Desarrollar un prototipo de asistente virtual capaz de:

- Identificar consultas frecuentes.
- Clasificar intenciones.
- Detectar mensajes de alerta.
- Escalar casos críticos a atención humana.
